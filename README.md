# Customer Management App

## 📌 Description
This is a simple Customer Management App built using JavaScript.  
It allows users to add, edit, and delete customer names.  
The data is stored in browser using localStorage, so it remains even after refreshing the page.

---

## 🚀 Features
- Add new customers
- Edit existing customers
- Delete customers
- Data persistence using localStorage

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript (DOM manipulation)

---

## 💡 What I Learned
- How to manipulate DOM using JavaScript
- Handling user input
- Working with arrays for data storage
- Using localStorage for saving data
- Event handling (click events)

---

## ▶️ How to Run
1. Download or clone this repository
2. Open `index.html` in browser
